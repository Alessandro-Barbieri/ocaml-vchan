Module Vchan_lwt_unix

module Vchan_lwt_unix: sig .. end
open_client domid port ?buffer_size () creates a client connection to the server running on domid with port port. This call will block until communication is established and it is safe to pass traffic. The underlying vchan connection will be disconnected when the input_channel is closed. If a ?buffer_size is given then 4 buffers of this size will be created: 2 for reading (vchan + Lwt_io) and 2 for writing.
